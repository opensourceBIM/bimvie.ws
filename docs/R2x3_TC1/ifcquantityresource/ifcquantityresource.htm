IFCQUANTITYRESOURCE

The data model consists of the following:

9  Entities

The IfcQuantiyResource defines a set of basic quantities that can be associated with products through the IfcElementQuantity (defined in the IfcProductExtension). The basic quantities are used to assign quantities, which have a particular meaning within a referenced method of measurement.

HISTORY: New schema in IFC 2x.
