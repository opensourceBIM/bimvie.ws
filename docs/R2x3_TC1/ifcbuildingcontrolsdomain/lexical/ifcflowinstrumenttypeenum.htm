IfcFlowInstrumentTypeEnum

Definition from IAI: The IfcFlowInstrumentTypeEnum defines the range of different types of flow instrument that can be specified.

HISTORY: New type in IFC 2x2

Enumeration

Value Definition Pset
PRESSUREGAUGE A device that reads and displays a pressure value at a point or the pressure difference between two points. YES
THERMOMETER A device that reads and displays a temperature value at a point. YES
AMMETER A device that reads and displays the current flow in a circuit. NO
FREQUENCYMETER A device that reads and displays the electrical frequency of an alternating current circuit. NO
PHASEANGLEMETER A device that reads and displays the phase angle of a phase in a polyphase electrical circuit. NO
POWERFACTORMETER A device that reads and displays the power factor of an electrical circuit. NO
VOLTMETER_PEAK A device that reads and displays the peak voltage in an electrical circuit. NO
VOLTMETER_RMS A device that reads and displays the RMS (mean) voltage in an electrical circuit. NO
USERDEFINED
NOTDEFINED
