IfcPreDefinedColour

Definition from IAI: The pre defined colour determines those qualified names which can be used to identify a colour that is in scope of the current data exchange specification (in contrary to colour specification which defines the colour directly by its colour components).

NOTE  Corresponding STEP name: pre_defined_colour. It has been made into an abstract entity in IFC. Please refer to ISO/IS 10303-46:1994, p. 141 for the final definition of the formal standard.
HISTORY  New entity in Release IFC2x Edition 2.

EXPRESS specification:

ENTITY IfcPreDefinedColour
ABSTRACT SUPERTYPE OF (IfcDraughtingPreDefinedColour)
SUBTYPE OF ( IfcPreDefinedItem);
END_ENTITY;
